# Coffee Companion MVP

## Features
- React frontend (Vite)
- Express backend
- PostgreSQL database
- Docker support

## Setup

### Backend
1. Create `.env` from `.env.example`
2. Run `node server.js` inside `server/`

### Frontend
1. Run `npm install` inside `client/`
2. Run `npm run dev`

### Docker
1. Run `docker-compose up` from root
